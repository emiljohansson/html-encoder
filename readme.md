# html-encoder

> Encodes html

## Usage

```js
const encode = require('path/to/html-encoder')
encode('<span>Hello</span>')
// => '&lt;span&gt;Hello&lt;/span&gt;'
```

## API

### html-encoder(string)

Encodes html

#### string

Type: `string`

HTML string to be encoded.
